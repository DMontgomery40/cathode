# Demo Brief: The Three-Click Video Machine

Use this as public-safe source material for the product demo.

## Goal

Explain that `Cathode` usually feels like a three-click workflow:

1. generate storyboard
2. generate everything
3. render

The deeper editing tools are important, but they should be framed as optional precision tools, not as the default experience.

## Audience

Marketers, founders, operators, SEO teams, consultants, and creators who want polished video output without a slow production pipeline.

## Core Story

This tool started as a highly specific internal workflow, then turned out to be broadly useful.

The real magic is not just that it can be edited deeply.

The real magic is that most of the time you do not need to edit it deeply at all.

The default experience is:
- paste rough notes, source text, or a final script
- click generate storyboard
- click the batch button that generates the assets
- click render

That is it.

Most of the time, it just works.

If one image or one line needs attention, you can open a single scene, preview it, regenerate one thing, or edit an existing image directly with Qwen image edit. That power matters, but it is not the main point. The main point is how often you do not need it.

## Messages To Land Early

- rough thoughts in, polished storyboard out
- usually three clicks from input to finished video
- scene-by-scene editing exists when you need it, not as required overhead
- local-first workflow with transparent project files
- simple env-driven provider setup instead of a cluttered control panel
- mix AI slides with uploaded clips or product footage
- works for short clips and scales to long-form projects

## Capability Notes

- visuals can be AI-generated slides, uploaded stills, uploaded clips, or mixed together
- voice can stay local or switch to cloud providers depending on what keys are present
- if one image is close but not perfect, Qwen image edit makes surgical fixes possible without starting over
- if something is wrong, the fix is usually one fast targeted change, not a full rebuild

## Future Note

Frame that as upcoming:

## Tone

- confident
- a little amazed by how well it works
- clean and premium, not salesy
- practical, not hypey
- should feel like someone who actually uses the tool all the time is showing you the fastest real workflow

## Visual Direction

- dark-mode product film energy
- blue-teal and warm amber accents
- cinematic but usable
- should show both the one-shot path and one expanded scene with preview/edit controls so the viewer understands both

## Ending

Land on this idea:

You do not need a giant production workflow to make something impressive.

Most of the time, this thing is just rough thoughts in, polished video out.
