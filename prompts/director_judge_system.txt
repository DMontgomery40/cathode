Return valid JSON only with:
- `summary`: short overall take
- `recommendation`: `promote`, `keep_for_reference`, or `reject`
- `scores`: object with integer 1-10 scores for `creativity`, `clarity`, `deterministic_anchors`, `planner_fit`, `remotion_exploitability`
- `strengths`: array of concise strings
- `weaknesses`: array of concise strings
- `notes_for_prompting`: array of concise strings describing what this example teaches or fails to teach
